adaboost算法

发表于 2015-05-24   |   分类于 ml   |  

ensemble方法都是用多个弱分类器组合成强分类器，只要弱分类器准确率大于50%即可（也就是随机。。）

ensemble主要分为bagging和boosting，前者分类器权重相同，后者分类器权重不同。前者的代表是random forest，后者是Adaboost。

原理

Adaboost的原理非常简单，每次迭代都根据分类器的正确率来确定分类器的权重，通过分类器的权重，重新计算样本的权重，达到迭代次数上限或者精度无法再变化就停止迭代。

计算

对于第t次迭代：

1、确定分类器的分类错误率$\epsilon$ = 错误分类样本$\ast D_{t-1}$/所有样本$\ast D_{t-1}$，从而可以确定分类器权重：
$$\alpha = \frac{1}{2} \ln(\frac{1-\epsilon}{\epsilon})$$

2、根据分类器权重，确定样本权重，加入有m个样本，初始样本权重为：$D_i = \frac{1}{m}$，迭代的权重为：$$D_t(i) = \frac{D_t(i) * E_t(i)}{Z_t}$$
$E_t(i)$的值，当样本在分类器上分类正确时，等于$e^{-\alpha_t}$，分类错误时等于$e^{\alpha_t}$, $Z_t$只为标准化$D_t$.

优劣

优势

  • 不用调参（除了迭代次数）
  • 简单，速度快
  • 精度不错，适合做baseline
  • 弱分类器不需要先验知识

劣势

  • 弱分类器如果过于复杂，容易overfitting
  • 弱分类器太弱也会导致overfitting
  • 对于均匀噪声（uniform noise）比较脆弱。

Linear Regression part one

发表于 2015-05-08   |   分类于 ml   |  

问题

考虑一个线性回归问题：
$$y = \theta_0 + \theta_1 x_1 + \theta_2 x_2 $$

假设$x_0 = 1$，因此可以以矩阵形式改写为：

$$Y = \Theta^T X$$

这个问题的损失函数可表述为：
$$J(\theta) = \sum_{i=0}^n(\theta_i x_i - y_i)^2$$

而线性回归问题可以转化成求解损失函数最小的问题。即：
$$\min_{\rm \theta} J(\theta) = \min_{\rm \theta} \sum_{i=0}^n(\theta_i x_i - y_i)^2$$

也就是求使得$J(\Theta)$最小的$\Theta$的值。

解法

梯度下降法

批量梯度下降法

梯度下降法是最直观的方法，通过不断的修正$\theta$来降低损失函数$J(\theta)$至最小。

梯度下降的原理就是$J(\theta)$对$\theta$进行求导，得到当前的梯度，也就是下降最快的斜率，然后修正$\theta$。
$$\Theta = \Theta - \gamma \frac{dJ}{d\theta}$$

随机梯度下降法

批量梯度下降法每次计算时计算所有样本，计算压力较大，因此就有随机梯度下降法。

在求导时，只随机选取一个样本进行计算，然后迭代。

局部梯度下降法

有时候问题并不是线性的，如果使用线性回归并不能很好的拟合，使用2次或者更高次的模型，可能造成overfitting。

如果数据量足够的话，可以在构建损失函数的时候，考虑需要预测的点附近的样本，增加他们的权重，将样本局部放大后有可能是线性的。

损失函数可以为：

$$J(\theta) = \sum_{i=0}^nw_i(\theta_i x_i -y_i)^2$$

相比普通的线性回归为题，增加的局部加权参数为：$w_i = exp(-\frac{(x-x_i)^2}{2\Gamma^2})$

这个参数取值范围为0到1，离所求的点近的点更加接近于1，公式很像正态分布的公式，但是没有任何关系，不过图形是类似的，离得越远，下降的越剧烈，这个是由参数$\Gamma$所决定的。

这个算法的好处是不需要线性也能解，前提是有足够的数据量。

牛顿法

牛顿法

牛顿法在逻辑回归里用的比较多，顺便介绍一下这个方法。

在求解$f(x)=0$时，首先选取接近这个解的点$x_1$，然后求导$f^{\prime}(x_1)$，导数的线与x轴相交于$x_2$，然后对点$(x_2,f(x_2)$求导，依次迭代，逐渐逼近最终解。

$$x = x - \frac{f(x)}{f^{\prime}(x)}$$

牛顿法应用到回归问题，我们可以将求损失函数最小转化为求解损失函数的一阶导为0的解。
即$J^{\prime}(\theta) = 0$

迭代公式为：

$$\Theta = \Theta - H^{-1} * \nabla_\theta J(\theta)$$

$\nabla_\theta J(\theta)$是一阶导，$H$是二阶导，也就是海森矩阵。

牛顿法能保证每次迭代后的点更加接近解，速度也比较快，因为是二次求导，曲线逼近解，而梯度下降是一次求导，直线搜索解。

牛顿法要求必须是凸的，而且二次可导，另外非常容易收敛至局部最小，对初始点特别敏感。

海森矩阵的计算比较复杂，因此虽然牛顿法逼近比较快，但是计算量仍然非常大。

海森矩阵如果非正定，没有解。。。。

拟牛顿法

上面也说过海森矩阵比较麻烦，不光是计算麻烦，前提条件也麻烦，因此开发了拟牛顿法——模拟一个近似的H矩阵，满足正定的条件。

构造近似矩阵方法的不同，对应不同的拟牛顿法，当前用的最广的是L-BFGS，很多大型广告系统使用的都是逻辑回归，并使用L-BFGS进行优化。
