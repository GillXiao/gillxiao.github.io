读书笔记-证析

发表于 2015-07-28   |   分类于 books   |  

缘起

读书第二周准备读这本书，做完计划才发现放在原公司，拖到这周才拿到手，也因此交了罚款。

这次去公司搬东西回来，发现最多的还是书，家里已经够多书了，已然成为搬家最大的烦恼，因此也想借机把书看完送掉，也算是最近流行的“断舍离”。

第一周看完设计书后就想看本和工作相关的书，然后就想起了这本：证析。

接触这本书源于新浪媒介策略总监的推荐，还写了几条书评，评价特别高。于是上市后第一时间购入，68的原价也算是大出血。可是看了几页后就看不下去，作者开篇花了好几页的篇幅写analytics为什么要翻译成证析，当时实在是受不了这种抠名字来秀智商的优越，于是就放弃了。

直到这周末重新拿起，才知道自己错过了什么。

推荐

开篇明义，强烈推荐：

  • 推荐所有和数据打交道的人阅读此书，不管是什么角色（从业者、决策者）。
  • 两年前大数据是热点，现在的大数据是臭虫，此书虽然副标题提到了大数据，但是不是。
  • 虽然这本书已经出版接近三年，但是仍未过时，章章经典，但是有些章节需要特定经历才能体会。
  • 这不是本技术书，对于被大数据概念轰炸了好几年的互联网人士来说，无阅读门槛。
  • 我准备再读一遍，不送人了，断舍离失败。

特点

这本书特点非常鲜明：

  • 这本书包含茫茫多案例，很多都是我闻所未闻！搞ppt创业的可以从里面摘了，和啤酒尿布say byebye。
  • 作者有自己的视角，有自己的洞察，并不是纯粹的堆砌案例。
  • 高屋建瓴，视角比较high level，能够给人一个全面的视角。

总之，没有丰富的经历和多年的经验，不可能写出这样的良心之作。

作者将这本书分为上下两部分，上编介绍数字化证据如何指导商业决策的背景、思想、应用；下编探讨争气在企业取得成功的关键要素：战略、组织、考核和决策流程。

小心：决策的误区

近年来，大数据思维一直在轰炸着我们的视线，数据决策思维逐渐占据越来越重要的地位，但是，相信直觉、经验、专家这些沿用了成百上千年的决策方式仍然是主流。

专家

先让我们看下专家的话：

根据南方基金风险策略部总监刘治平统计的数据显示：2007年1月1日至2009年12月8日，纳入统计范围的130只偏股型基金中每年的业绩均排名前30%、前40%、前50%的基金仅分别有3只、6只、12只。。。。。。值得一提的是，仅有六只连续三年进入业绩前40%的基金大多为明星基金经理掌控。

是不是对这几只基金非常崇拜？是不是巨牛逼？如果潜下心来算一下，你会发现这和随机是一样的。

基金经理却可能认为自己的成功来自于自己投资策略的成功，而不是运气。

理论

82年出版的《追求卓越》一书售出超过600万册，算是管理界的经典，作者通过六个指标筛选出了62家企业，进一步让行业专家进行创新性评估，筛除43家企业，认为他们是卓越企业，并从中归纳了企业成功的八项特质。摆数据、讲道理，看起来非常令人信服。

实际上，自从此书出版，这43家的业绩就开始走下坡路，随后的五年62%的企业低于市场平均投资回报。最后作者也承认伪造了数据。

方法

四张扑克牌：E、K、4、7。某人告诉你存在一个规律：“如果某张扑克牌的一面是元音字母，那么它的另外一面一定是个偶数”，如果你想验证这个说法，需要翻开哪两张扑克牌？

要想不被各种骗（成功学、专家、大师），需要参透此题所反映的方法论。

读书笔记：写给大家看的设计书

发表于 2015-07-15   |   分类于 books   |  

写在前面

一直想读书，可惜都没沉浸下来，跑去写代码自娱自乐去了，直到朋友用美女来勾引我进了一个每周一本书的群。

每周一本，没看完罚款28，钱是小事，面子是大事，美女还没怎么遇见，催更的鼓声天天在耳边响起，真是自作孽。。

至少博客算是用上了吧。

下面是正文。。。。

作为一个典型的理工男，非常羡慕那些会画画会写作的人，也一直为自己没有丝毫的设计能力而感伤，如果连设计都会，岂不是能搞搞全栈了？

直到看到这本书：写给大家看的设计书，也许英文名称表达的更加清晰：The Non-Designer’s Design Book，给业余选手看的设计书。

四大基本原则

设计的四大基本原则包括：对比、重复、亲密性、对齐。

这四大原则非常简单易懂，立马能提高设计档次。

  • 亲密性主要是使内容能够进行区分，不同类别、重要性的内容要有不同的距离。
  • 对齐主要说明整体设计要有明显的对齐的轴，尽量别用居中。
  • 重复和对比使你能够增加内容的区分度。

颜色

我最烦的就是颜色，对颜色的匹配完全没有感觉，写PPT最喜欢用微软默认的几个颜色，后来看到一篇文章说，那是最土的配色，因为所有人都在用。。。。。。实际上我现在脑海里呈现的是新浪PPT模板里那倔强的屎黄。

本书作者在讲颜色的时候首先讲了色轮，色轮中距离最远的颜色对比性更强，更值得组合运用。

在此基础上再引入了暗色、亮色、色质、暖色、冷色。组合其来更加丰富。

看完这章，PPT的颜色搭配是不用愁了。。。

字体

字体没太大兴趣，没怎么看。

总结

设计技巧都是服务于需求。关键在于捋顺自己的需求点：你想给读者传达什么？根据这个创建有强烈对比的中心点。

接下来对信息分组，按照逻辑来决定亲密性。在页面组织方面，要建立并维护明确的对齐。找出可重复项并进行重复。

设计和写报告一样，重思路和逻辑，后面的其他技巧都是服务于这个，并非灵魂。

写在最后

书评很简单，貌似草草了事，实际上，这本书将长期在我的手边，以便随时翻阅。

天赋只是决定了天花板的高低，入门加进阶并不那么难，不应再给自己任何拒绝学习的理由。

下周看证析，一定要好好写笔记。。。。。

恩，希望如此。

adaboost算法

发表于 2015-05-24   |   分类于 ml   |  

ensemble方法都是用多个弱分类器组合成强分类器，只要弱分类器准确率大于50%即可（也就是随机。。）

ensemble主要分为bagging和boosting，前者分类器权重相同，后者分类器权重不同。前者的代表是random forest，后者是Adaboost。

原理

Adaboost的原理非常简单，每次迭代都根据分类器的正确率来确定分类器的权重，通过分类器的权重，重新计算样本的权重，达到迭代次数上限或者精度无法再变化就停止迭代。

计算

对于第t+1次迭代：

1、确定分类器的分类错误率$\epsilon$ = 错误分类样本$\ast D_{t}$/所有样本$\ast D_{t}$，从而可以确定分类器权重：
$$\alpha = \frac{1}{2} \ln(\frac{1-\epsilon}{\epsilon})$$

2、根据分类器权重，确定样本权重，加入有m个样本，初始样本权重为：$D_0 = \frac{1}{m}$，迭代的权重为：$$D_{t+1}(i) = \frac{D_t(i) * E_t(i)}{Z_t}$$
$E_t(i)$的值，当样本在分类器上分类正确时，等于$e^{-\alpha_t}$，分类错误时等于$e^{\alpha_t}$, $Z_t$只为标准化$D_t$.

优劣

优势

  • 不用调参（除了迭代次数T）
  • 简单，速度快
  • 精度不错，适合做baseline
  • 弱分类器不需要先验知识

劣势

  • 弱分类器如果过于复杂，容易overfitting
  • 弱分类器太弱也会导致overfitting
  • 对于均匀噪声（uniform noise）比较脆弱。

Linear Regression part one

发表于 2015-05-08   |   分类于 ml   |  

问题

考虑一个线性回归问题：
$$y = \theta_0 + \theta_1 x_1 + \theta_2 x_2 $$

假设$x_0 = 1$，因此可以以矩阵形式改写为：

$$Y = \Theta^T X$$

这个问题的损失函数可表述为：
$$J(\theta) = \sum_{i=0}^n(\theta_i x_i - y_i)^2$$

而线性回归问题可以转化成求解损失函数最小的问题。即：
$$\min_{\rm \theta} J(\theta) = \min_{\rm \theta} \sum_{i=0}^n(\theta_i x_i - y_i)^2$$

也就是求使得$J(\Theta)$最小的$\Theta$的值。

解法

梯度下降法

批量梯度下降法

梯度下降法是最直观的方法，通过不断的修正$\theta$来降低损失函数$J(\theta)$至最小。

梯度下降的原理就是$J(\theta)$对$\theta$进行求导，得到当前的梯度，也就是下降最快的斜率，然后修正$\theta$。
$$\Theta = \Theta - \gamma \frac{dJ}{d\theta}$$

随机梯度下降法

批量梯度下降法每次计算时计算所有样本，计算压力较大，因此就有随机梯度下降法。

在求导时，只随机选取一个样本进行计算，然后迭代。

局部梯度下降法

有时候问题并不是线性的，如果使用线性回归并不能很好的拟合，使用2次或者更高次的模型，可能造成overfitting。

如果数据量足够的话，可以在构建损失函数的时候，考虑需要预测的点附近的样本，增加他们的权重，将样本局部放大后有可能是线性的。

损失函数可以为：

$$J(\theta) = \sum_{i=0}^nw_i(\theta_i x_i -y_i)^2$$

相比普通的线性回归为题，增加的局部加权参数为：$w_i = exp(-\frac{(x-x_i)^2}{2\Gamma^2})$

这个参数取值范围为0到1，离所求的点近的点更加接近于1，公式很像正态分布的公式，但是没有任何关系，不过图形是类似的，离得越远，下降的越剧烈，这个是由参数$\Gamma$所决定的。

这个算法的好处是不需要线性也能解，前提是有足够的数据量。

牛顿法

牛顿法

牛顿法在逻辑回归里用的比较多，顺便介绍一下这个方法。

在求解$f(x)=0$时，首先选取接近这个解的点$x_1$，然后求导$f^{\prime}(x_1)$，导数的线与x轴相交于$x_2$，然后对点$(x_2,f(x_2)$求导，依次迭代，逐渐逼近最终解。

$$x = x - \frac{f(x)}{f^{\prime}(x)}$$

牛顿法应用到回归问题，我们可以将求损失函数最小转化为求解损失函数的一阶导为0的解。
即$J^{\prime}(\theta) = 0$

迭代公式为：

$$\Theta = \Theta - H^{-1} * \nabla_\theta J(\theta)$$

$\nabla_\theta J(\theta)$是一阶导，$H$是二阶导，也就是海森矩阵。

牛顿法能保证每次迭代后的点更加接近解，速度也比较快，因为是二次求导，曲线逼近解，而梯度下降是一次求导，直线搜索解。

牛顿法要求必须是凸的，而且二次可导，另外非常容易收敛至局部最小，对初始点特别敏感。

海森矩阵的计算比较复杂，因此虽然牛顿法逼近比较快，但是计算量仍然非常大。

海森矩阵如果非正定，没有解。。。。

拟牛顿法

上面也说过海森矩阵比较麻烦，不光是计算麻烦，前提条件也麻烦，因此开发了拟牛顿法——模拟一个近似的H矩阵，满足正定的条件。

构造近似矩阵方法的不同，对应不同的拟牛顿法，当前用的最广的是L-BFGS，很多大型广告系统使用的都是逻辑回归，并使用L-BFGS进行优化。
